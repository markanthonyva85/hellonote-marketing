# Microsoft Clarity Weekly Reports

This folder stores all weekly Microsoft Clarity data and analysis reports for HelloNote EMR.

## Weekly File Naming Convention
- clarity-week-01-baseline.md
- clarity-week-02-report.md
- clarity-week-03-report.md

## What Each Weekly Report Tracks
- Total sessions and unique users
- Top pages by traffic and engagement
- Heatmap insights and top click areas
- Rage clicks and dead clicks
- Scroll depth per key page
- Week over week comparison
- SEO and content recommendations based on behavior data
